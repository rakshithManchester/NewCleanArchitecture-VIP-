# NewCleanArchitecture


Sample application MVVMVGSeries has Three screens 
  a) Login b) Home c) Detail.

  a) Login Folders : 
         : LoginViewController
         : LoginConfigurator
         : LoginRouter
         : Login(Storyboard)
         : LoginViewModel
         : LoginWorker
         : LoginPresenter
         : LoginInteractor
         
b) Home Folders
         : HomeViewController
         : HomeConfigurator
         : HomeRouter
         : Home(Storyboard)
         : HomeViewModel
         : HomeWorker
         : HomePresenter
         : HomeInteractor
         : WeatherTableViewCell
         : WeatherForecastBusinessModel
         : WeatherForecast
         
c) Detail 
         : DetailViewController
         : DetailConfigurator
         : DetailRouter
         : Detail(Storyboard)
         : DetailViewModel
         : DetailWorker
         : DetailPresenter
         : DetailInteractor

Common files are - Coordinator
                   Utils
                   StoryBoarded
                   SceneDelegate
                   AppDelegate
